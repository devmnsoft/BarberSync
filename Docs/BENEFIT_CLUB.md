# BENEFIT_CLUB

Módulo inicial do ecossistema de experiência do cliente final do BarberSync.

## Escopo
- Modelagem SQL dedicada no diretório `ScriptsSQL`.
- Estrutura pronta para endpoints e serviços na API.
- Preparado para evolução incremental com tenant e branch.

## Entregas desta fase
- Tabelas principais.
- Regras-base de relacionamento.
- Seeds de demonstração.
- Views para consumo no dashboard CX.
