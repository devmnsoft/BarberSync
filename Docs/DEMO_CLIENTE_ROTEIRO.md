# Roteiro de Demonstração ao Cliente — BarberSync 2.0

1. Abrir PublicWeb.
2. Mostrar a landing page e proposta de valor.
3. Simular agendamento público.
4. Entrar no AdminWeb.
5. Mostrar dashboard gerencial.
6. Cadastrar cliente.
7. Cadastrar serviço.
8. Criar agendamento.
9. Fazer check-in do cliente.
10. Abrir comanda.
11. Adicionar serviço/produto.
12. Registrar pagamento.
13. Mostrar caixa e fechamento parcial.
14. Mostrar estoque crítico.
15. Perguntar ao Copilot (insight de vendas/agenda).
16. Abrir Totem.
17. Escolher serviço no Totem.
18. Efetuar pagamento mockado.
19. Mostrar tela de sucesso.
20. Encerrar com relatórios e próximos passos.
