# Manual de Uso Admin

Use o menu lateral para acessar SaaS Control Center, Lead to Cash, Channel Manager, Dashboard, Operations, Cliente 360, PDV, Caixa, Estoque, Campanhas, Copilot e Reports. Todas as ações demo atualizam DemoStore v6 e emitem eventos auditáveis no EventBus.

## Operação diária
1. Carregue um cenário no SaaS Control Center.
2. Use Lead to Cash para criar movimento.
3. Use Operations para mudar status dos agendamentos.
4. Use ServiceOrders e Cash para fechar venda.
5. Verifique Stock, Campaigns, Clients e Dashboard.
